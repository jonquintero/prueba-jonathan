## Spa Jonathan

Simple CRUD realizado con Laravel y Vue.js

- Descargar o clonar el repositorio.
- Copiar el archivo de entorno cp .env.example .env
- Realizar composer install.
- Ejecutar npm install --save vue-axios vue-router
- Crear en la carpeta database el archivo database.sqlite
